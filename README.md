# Trusted-installer-shell
Launches a command prompt with trusted installer permissions. 

### Build requirements:

#### **MSVS C++ compiler (VS v17+)**
#### **Windows SDK 8.1** 




**Don't run it on anything below Windows XP; Trusted Installer didn't exist back then.** 

*I only compiled it with VS 18 and the latest Windows SDK, but apparently the ones I mentioned are supported*

*Just download the C++ file and compile it with Visual Studio, not that complicated and if you have issues, ask AI.*
